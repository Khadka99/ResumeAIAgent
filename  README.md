# Resume AI Agent

An AI-powered desktop application that customizes resumes and generates cover letters based on job descriptions.

## Features (Planned)

- Resume customization
- Cover letter generation
- ATS keyword analysis
- OpenAI support
- Ollama local model support
- ChromaDB knowledge base
- Desktop GUI (CustomTkinter)
- Word & PDF export

## Tech Stack

- Python
- OpenAI API
- Ollama
- LangGraph
- ChromaDB
- CustomTkinter
- Git & GitHub
